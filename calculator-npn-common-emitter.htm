Brief for NPN Common Emmiter arrangement :

Before proceed reading see circuit diagram at end of page!

On collector we have a current "IC" causing a voltage drop "IC*RC" where "IC" is "IB" multiplied by "hFE", on base we have a current "I RB" causing voltage drop to "VB" that need to be "VS" + "VE" + 0.6, on emitter we have a current "IB" plus "IC" causing a voltage drop on "RE" that need to be "VE"; for more gain we have "CE" 1/10th of "RE".
